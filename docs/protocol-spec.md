# AquaIntelligence 通信协议规范

## Modbus-RTU 寄存器映射

| 地址 | 指标 | 类型 | 单位 | 缩放 | 说明 |
|------|------|------|------|------|------|
| 0x0001 | DO | UINT16 | mg/L | 0.01 | 实际值=寄存器值/100 |
| 0x0002 | 水温 | INT16 | ℃ | 0.1 | 实际值=值/10 |
| 0x0003 | pH | UINT16 | pH | 0.01 | 范围 0.00-14.00 |
| 0x0004 | 设备电流 | UINT16 | A | 0.1 | 增氧机负载判断 |

## WebSocket 二进制帧结构

```
┌────────┬──────────┬──────────┬──────────┬──────────┬──────────┐
│ 帧头   │ 时间戳    │ 设备ID   │ 负载类型  │ Payload  │ HMAC签名  │
│ 2 Byte │ 8 Byte   │ 4 Byte   │ 1 Byte   │ N Byte   │ 32 Byte  │
├────────┼──────────┼──────────┼──────────┼──────────┼──────────┤
│0xAA55  │ 纳秒Unix │ 网关UID  │ 0x01实时  │ 加密数据  │ SHA-256  │
│        │          │          │ 0x02控制  │          │          │
│        │          │          │ 0x03心跳  │          │          │
│        │          │          │ 0x04 ACK  │          │          │
└────────┴──────────┴──────────┴──────────┴──────────┴──────────┘
```

## 安全机制
- 控制指令：HMAC-SHA256 签名 + 64位 Nonce + 2000ms 时间窗口
- 传输层：TLS 1.3 全链路加密
- 认证：OAuth 2.0 + JWT
- 防重放：Nonce 去重缓存 + 时间戳校验
