# AquaIntelligence 系统架构设计

> 高价值水产全生命周期数字孪生平台

## 云-边-端 三位一体架构

### 感知层（物理端）
- 传感器：DO, pH, Temp, ORP, NH3-N, 电导率
- 协议：Modbus-RTU (RS485), 100ms 轮询, 10Hz 采样
- 硬件：工业级 ARM 网关，双网冗余，UPS 后备电源

### 边缘计算层
- 数据清洗：中值滤波 + 3σ 异常剔除
- 本地自治：网络中断 500ms 内切换 PID 控制
- 断点续传：24h 本地缓存 + 背压补录

### 云端中枢层
- WebSocket 引擎：10k 并发, 50k msg/s
- AI 预测：LSTM 2h 溶解氧趋势预测
- 规则引擎：多参数耦合分析 + 风险评级
- 存储：Redis(快照) + MySQL(关系) + InfluxDB(时序)

### 展现层（终端）
- Web 3D 指挥中心：Three.js 数字孪生
- 移动端 App：红绿灯仪表盘 + 应急控制

## 延迟预算 (200ms E2E)
| 阶段 | 预算 |
|------|------|
| 边缘采集与预处理 | <= 30ms |
| 网络传输 (WebSocket) | <= 50ms |
| 云端逻辑与 AI 推演 | <= 70ms |
| 前端双缓冲渲染 | <= 50ms |
